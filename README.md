# Morning Brief

A personal AI-powered news digest for The Hacker News, BleepingComputer, and Risky Business.

## Setup (one-time)

### 1. Install Node.js
If you don't have it: https://nodejs.org — download the "LTS" version and install it.

### 2. Add your Anthropic API key
- Rename `.env.example` to `.env`
- Go to https://console.anthropic.com/settings/keys and create a key
- Paste it into `.env` so it reads: `ANTHROPIC_API_KEY=sk-ant-...`

### 3. Install dependencies
Open a terminal in this folder and run:
```
npm install
```
This downloads the React and Vite packages listed in package.json. Only needed once.

## Running the app

```
npm run dev
```

Then open http://localhost:5173 in your browser.

To stop the app, press Ctrl+C in the terminal.

## How it works

- **Feeds** are fetched via rss2json.com, which converts RSS/XML to JSON
- **AI summaries** are generated by Claude via the Anthropic API
- **"Previously read"** tracking uses localStorage — stored in your browser, persists across sessions
- Your API key lives only in `.env` and is never sent to the browser
